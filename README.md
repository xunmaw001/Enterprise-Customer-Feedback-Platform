# Enterprise-Customer-Feedback-Platform
基于 Spring Boot + Vue 的企业客户信息反馈平台，旨在加强企业与客户间的沟通。该平台支持客户提交反馈信息，如意见、建议和投诉等，同时企业可查看并处理这些信息。前端 Vue 提供友好的交互界面，操作方便；后端 Spring Boot 确保系统稳定，保证数据安全存储与高效处理，有助于企业提升服务质量，优化客户体验。
